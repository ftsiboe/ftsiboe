# Research-portfolio metrics

Generated by `data-raw/scripts/build_research_metrics.R` (with plotting helper
`research_metrics_plots.R`). Run it from the repo root:

```bash
Rscript data-raw/scripts/build_research_metrics.R
```

It enriches your publication list and produces two things: a **theme-evolution**
view (JEL codes + keywords over time) and a **collective impact** score
(citation-weighted average of journal impact factors).

## What it needs (one-time installs)

```r
install.packages(c("jsonlite","pdftools","ggplot2","ggstream","ggwordcloud"))
```

`jsonlite` is required. `pdftools` reads JEL/keywords from your PDFs. The three
plotting packages are optional — without them the data still builds and the
plots degrade gracefully (a message tells you which to install).

## Data sources

- **Citations + journal** — [OpenAlex](https://openalex.org) by DOI (free, no key).
- **Impact factor** — OpenAlex *2-year mean citedness*, an **open, automatable
  proxy** for the Clarivate JCR Impact Factor (the real JCR number is paywalled
  and has no free API). To use official IFs, see `if_overrides.csv` below.
- **JEL codes + keywords** — read straight from the **author-assigned lines in
  your local PDFs**. About 20 papers print JEL (the econ journals); non-econ
  outlets (PLOS ONE, Agronomy Journal, Food Control, …) have none — fill those
  via `jel_overrides.csv`.

## Outputs

| File | What |
|---|---|
| `publications_enriched.csv` | one row per paper: year, journal, citations, IF proxy, JEL, keywords |
| `collective_impact_by_year.csv` | citation-weighted IF trajectory |
| `themes_by_year.csv` | JEL/keyword counts by year (feeds the three theme plots) |
| `../../docs/assets/research-metrics/collective_impact.png` | the IF trajectory |
| `…/themes_streamgraph_jel.png`, `…_keyword.png` | themes flowing over time (JEL and keywords separate) |
| `…/themes_bubble_jel.png`, `…_keyword.png` | year × term bubble timeline |
| `…/themes_wordclouds_jel.png`, `…_keyword.png` | word clouds per era |

## Optional overrides (header-only templates are here; fill as needed)

- **`jel_overrides.csv`** — `doi,jel` — JEL codes the PDF parser missed
  (`;`-separated, e.g. `10.1016/j.foodpol.2023.102505,Q12;Q18;G22`).
- **`keyword_overrides.csv`** — `doi,keywords` — replace auto keywords
  (`;`-separated).
- **`if_overrides.csv`** — `issn,year,impact_factor` — official JCR IFs by
  journal; if a paper's journal ISSN matches, the override (most recent year)
  replaces the OpenAlex proxy.

## The collective-impact formula

For each year *Y*: **Collective IF(Y) = Σ(IFᵢ · Cᵢ(Y)) / Σ Cᵢ(Y)** over papers
published through *Y*, where *Cᵢ(Y)* is paper *i*'s cumulative citations through
*Y*. It measures the citation-weighted prestige-reach of the active portfolio.
Journal IF is held at its current value (a full historical per-year JCR series
isn't reconstructable from free data); supply `if_overrides.csv` for true
year-by-year IFs.

`cache/` holds raw OpenAlex responses so re-runs are fast — it's git-ignored;
delete it to force a full refresh.
